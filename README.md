# Tagalong Releases

Official releases for [Tagalong](https://tagalongai.com) - AI-powered meeting transcription and summarization for macOS.

## Download

**Latest Version:** [Download Tagalong.dmg](https://github.com/tagalong-ai/releases/releases/latest/download/Tagalong.dmg)

## Installation

1. Download the DMG file
2. Open the DMG and drag Tagalong to Applications
3. Launch from Applications folder
4. Grant microphone permissions when prompted

## Requirements

- macOS 15.0 (Sequoia) or later
- 50 MB free disk space
